Hyundai UX Guideline
Asset Package & Style Guide 현대자동차 UX 가이드라인
에셋 패키지 & 스타일 가이드

Client

Hyundai Motor Company

Company

LISN Design Inc.

Role

Graphic Style Direction · Component Design · Style Guide · Documentation

Duration

Nov 2024 – Apr 2025 · 6 months

Category

Design System · Documentation

Overview프로젝트 개요

Built a scalable Figma-based graphic asset library for Hyundai Motor's UX guideline documentation — replacing a manual PowerPoint-based workflow and reducing production time by 75%. 현대자동차의 UX 가이드라인 제작에 사용되는 그래픽 에셋 라이브러리를 피그마 기반으로 구축하여, PPT 수작업 방식을 대체하고 작업 시간을 75% 단축했습니다.

Problem문제

UX principles were documented using manually drawn PowerPoint shapes — inconsistent, time-consuming, and hard for other departments to interpret correctly. Every new guideline required rebuilding illustrations from scratch. UX 원칙은 PPT 수작업 도형으로 문서화되었고, 일관성이 없고 시간이 오래 걸렸으며 타 부서의 해석 차이도 발생했습니다. 새 가이드라인마다 일러스트를 처음부터 다시 만들어야 했습니다.

Process과정

  • Classified vehicle UX components into a view-based hierarchy (front / side / quarter view) 차량 UX 구성요소를 시점(전면·측면·쿼터뷰)별로 분류
  • Developed a flat, minimal, line-based graphic style inspired by IKEA assembly manuals 이케아 설명서에서 영감을 받은 평면·미니멀·라인 기반 그래픽 스타일 정립
  • Built modular Figma components with Variant support for Do/Don't toggling Do/Don't를 Variant로 관리하는 모듈화 Figma 컴포넌트 구축
  • Created a style guide (color, line weight, spacing) for consistent asset production 일관된 에셋 제작을 위한 스타일 가이드(컬러·선 두께·간격) 작성
  • Designed visual, auditory, and haptic effect icons for multi-sensory UX expression 시각·청각·햅틱 효과 아이콘을 설계하여 다감각적 UX 표현 지원

Outcome성과

75%

Reduction in guideline production time (2 hours → 30 minutes per document) 가이드라인 1건 제작 시간 단축 (2시간 → 30분)

~180h

Saved annually based on 10 guidelines per month 월 10건 기준 연간 절약 시간

What I Learned회고

Good design systems aren't just beautiful — they're usable by others without explanation. 좋은 디자인 시스템은 예쁜 것이 아니라, 설명 없이도 누구나 쓸 수 있는 것입니다.
