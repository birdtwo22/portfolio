Client

UI/UX Design · Prototyping · Motion & Illustration · Developer Handoff

Duration

Nov 2023 – Apr 2024 · 6 months

Category

Fintech · Mobile UI

Overview프로젝트 개요

Samsung Card's flagship payment feature — the App Card — was migrated to Monimo, Samsung's integrated finance app. The goal was to redesign the experience from scratch to drive user adoption and simplify a complex payment flow. 삼성카드의 핵심 결제 기능인 앱카드를 삼성 금융 통합앱 모니모로 이관하며, 사용자 유입을 높이고 복잡한 결제 흐름을 개선하는 것이 목표였습니다.

Problem문제

The original App Card UI listed features in a flat, scrollable structure — key functions were buried, and onboarding was fragmented across multiple screens. Monimo's MAU was below 10% of total Samsung Card users. 기존 앱카드 화면은 기능이 나열식으로 구성되어 중요 기능이 스크롤 아래로 밀려 있었고, 사용 방법도 분산되어 진입장벽이 높았습니다. 모니모의 MAU는 삼성카드 전체 사용자의 10% 미만이었습니다.

Outcome성과

85%

A/B test with 20 users preferred the new Monimo UI 사용자 20명 A/B 테스트 결과 85%가 새 UI를 더 쉽다고 평가

490K

MAU increase within 3 months of launch (+9.9%) 출시 3개월 만에 MAU 49만 명 증가 (+9.9%)

Process과정

  • Benchmarked 8 competitor app card UIs for layout and clarity 8개 경쟁사 앱카드 화면을 레이아웃·직관성 기준으로 분석
  • Redesigned using grouped features + tab navigation to eliminate unnecessary scrolling 기능 그룹화 + 탭 내비게이션으로 불필요한 스크롤 제거
  • Reduced screen depth by enabling QR scan and code entry via direct tab access QR 스캔·코드 입력을 탭 직접 접근 방식으로 개선해 화면 뎁스 단축
  • Created an illustration-based motion guide to lower onboarding friction 일러스트 기반 모션 가이드로 온보딩 진입장벽을 낮춤
  • Added a branded coin-spin animation to the Monimo main screen to increase visual attention 모니모 메인 화면에 브랜드 코인 스핀 애니메이션을 추가해 시각적 주목도 향상

What I Learned회고

Complex features don't need to be simplified — they need to be reorganized. Good IA is invisible. 복잡한 기능을 단순하게 만드는 것이 아니라, 올바르게 재구성하는 것이 좋은 UI라는 걸 배웠습니다.
